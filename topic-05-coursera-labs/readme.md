# Coursera Labs — Programming with JavaScript

## What This Folder Covers

Real graded lab assignments from the Meta Frontend Developer Certificate
on Coursera. Each file contains the original instructions, my first attempt,
bugs I ran into, fixes, and a full reflection written after completion.

These are not clean solutions, they are honest records of the full process
including what broke, why it broke, and what I learned from fixing it.

## Labs

**Lab 01 — Little Lemon Receipt Maker**
Built two functions, getPrices() and getDiscount(), to simulate a
restaurant receipt system. Applied Red Green Refactor for the first time
on a real graded assignment. Covered arrays of objects, for loops,
defensive coding, nested conditionals, and parameter passing between functions.

## How These Labs Are Documented

Every lab follows the same structure:

- RED — written expectations before any code is written
- GREEN — minimum code to satisfy the requirements
- Bugs — what broke, why it broke, how it was fixed
- REFACTOR — what was cleaned up after it worked
- Reflection — honest thoughts on the process and what clicked
